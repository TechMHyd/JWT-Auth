All the samples are written in TypeScript (Angular 2/4). To run these applications follow these steps:

1. Open Command Prompt
2. Navigate to your project from cmd     like: cd flopKart
3. From cmd run $npm install to get    all dependencies from internet
4. Once all the dependencies are    downloaded run $npm start to run    the application in your browser


Prerequisites:

1. NodeJs 6.1 or above 
2. Open internet access to download the dependencies.
